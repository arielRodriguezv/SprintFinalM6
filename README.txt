Sprint final de modulo 6 / Proyecto Spring Boot

Proyecto Spring boot para la administracion de usuarios. El proyecto usa spring security para el manejo de sesiones contra base de datos. Las solicitudes hacia la base de datos se encuentran manejadas mediante JPA. Las vistas estan creadas en formato html y gestionadas mediante thymeleaf. El proyecto esta basado en modelo MVC, y sus dependencias estan gestionadas mediante Maven.
